# k8s-training-new

## Tools installation

### Docker

Install Docker according to the [installation instructions](https://docs.docker.com/engine/install/) specific for your operating system.

### minikube

Install Minikube according to the [installation instructions](https://minikube.sigs.k8s.io/docs/start/) specific for your operating system.


### kubectl

Install kubectl according to the [installation instructions](https://kubernetes.io/docs/tasks/tools/) specific for your operating system.
